Python 定向爬虫python

发表于 2019-09-07 更新于 2020-03-21 分类于 python

爬虫目标

  1. 爬取豆瓣电影的电影详情、剧情介绍
  2. 根据豆瓣所爬取的电影名称到电影天堂 寻找相应电影的下载链接ftp和磁性链接magnet
  3. 将所得数据存放于数据库中或文件中

爬虫功能

  1. URLMangage 管理可爬取URL链接，存放于MYSQL数据库中，用以判重，断点续爬
  2. URLcode URL解析，解析protocol协议，domain域名，path相对路径，以及判断url所对应的文件类型或目录
  3. Spider 爬虫主程序，通过给定的URL入口进行递归爬取

依赖第三方库

  1. requests HTTP请求
  2. BeautifulSoup HTML地方库
  3. lxml xml解析
  4. pymysql mysql的py库
